# Getting Started

The simplest and fastest way to install Klassco is by using a JavaScript
package manager.

<Tabs items={["pnpm", "bun", "npm", "yarn"]}>
  <Tabs.Tab>
      ```bash
      $ pnpm -g i klassco

      ```
  </Tabs.Tab>
  <Tabs.Tab>
      ```bash
      bun -g add klassco
      ```
  </Tabs.Tab>
  <Tabs.Tab>
      ```bash
      npm -g i klassco
      ```
  </Tabs.Tab>
  <Tabs.Tab>
      ```bash
      yarn -g add klassco
      ```
  </Tabs.Tab>
</Tabs>

**Note**: If you are using Windows, type `klassco-win` in your terminal
instead of `klassco`.

If you want to build from source, see the
[repo](https://github.com/cipherlogs/klassco) for guidance.
