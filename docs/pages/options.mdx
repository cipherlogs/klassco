# Core options
Klassco’s flags are designed to be
[composable](../core-concepts/key-features), which means that each flag can
produce different results depending on the other flags used.

This feature makes Klassco much more powerful than the standard Unix CLI. It’s
worth taking a few minutes to explore some examples and scenarios that might
not immediately come to mind, so you can learn how to use Klassco in new and
creative ways.
